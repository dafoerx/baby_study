## 添加 APK 打包支持
- **方案**: 原生 Android 项目（Java + WebView），不依赖 Node.js/Capacitor
- **Android 项目**: `android/` 目录，Gradle 7.4.2 + compileSdk 33 + minSdk 21
- **主Activity**: WebView 加载 `file:///android_asset/www/index.html`
- **包名**: `com.babystudy.flashcard`，应用名"安宝闪卡"
- **打包脚本**: `build-apk.sh` — 自动同步 Web 资源到 assets/www/ 并调用 Gradle 构建
- **PWA 支持**: 添加 manifest.json + sw.js，浏览器中也可"添加到桌面"
- **图标生成**: `generate-icons.py` — 纯 Python 生成 PNG 图标（红底白色5点）
- **打包前提**: 需安装 Android SDK（设置 ANDROID_HOME 环境变量）

## 完善一键打包（全自动，零依赖）
- **问题**: 环境只有 Java 8，AGP 7.4.2 需要 JDK 11+；且无 Android SDK
- **解决**: `build-apk.sh` 自动下载 JDK 11、Android SDK 组件、Gradle
- **自动下载**: JDK 11 (Adoptium Temurin) + platform-33 + build-tools-33.0.2 + Gradle 7.5.1
- **首次下载约 500MB**，之后复用缓存在 `.jdk11/`、`.android-sdk/`、`.gradle-dist/`
- **打包结果**: `output/baby-flashcard-debug.apk`，3.0MB
- **真正一键**: 只需 `./build-apk.sh`，无需预装任何依赖
